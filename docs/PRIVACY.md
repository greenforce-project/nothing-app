# Privacy Policy

**Last updated: July 2026**

## Overview

`nothing` is an offline entertainment application.

The application does not collect, transmit, share, sell, or
process personal information.

## Data stored on the device

The application stores only the following information locally:

- the user's best challenge time;
- the user's number of attempts.

This information is stored using Android SharedPreferences and
never leaves the user's device.

The locally stored statistics can be deleted from inside the
application by using the reset option.

## Internet access

The application does not request internet permission and does not
connect to online services.

## Analytics and advertising

The application does not use:

- analytics services;
- advertising networks;
- tracking technologies;
- user accounts;
- cloud storage;
- third-party data collection services.

## Permissions

The application does not request sensitive Android permissions.

## Children's privacy

The application does not knowingly collect information from
children or any other users.

## Changes to this policy

This policy may be updated if the application's functionality
changes. The latest version will be available in this repository.
